## Step 4: Native Features

```
Tell Tanjiro to implement these Capacitor native features:
1. Camera — take progress photos after workouts
2. Geolocation — track outdoor running routes
3. Push Notifications — daily workout reminders
4. Local Storage — offline workout logging with sync
```

Tanjiro creates custom hooks for each:

```typescript
// src/hooks/useCamera.ts
import { Camera, CameraResultType } from '@capacitor/camera';

export function useCamera() {
  const takePhoto = async () => {
    const photo = await Camera.getPhoto({
      resultType: CameraResultType.Uri,
      quality: 80,
    });
    return photo.webPath;
  };
  return { takePhoto };
}
```

Each hook works on both web (browser fallback) and native (Capacitor plugin).
